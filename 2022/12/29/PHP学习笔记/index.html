三、函数

explode()函数

字符串分割函数

格式:

array explode(string $separator, string $str [, int $limit])

  1. 第一个参数为分隔符

  2. 第二个参数为待分割的字符串

  3. 第三个为可选参数:

    • 正数: 为几就分割几个, 剩余的都存放在最后一个

    • 零: 会被当做1, 输出来也就是原先的字符串

    • 负数: 从列表后边减, -1就是去掉列表最后一个
